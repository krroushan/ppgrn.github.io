PhonePe Payment Gateway Integration

PhonePe Payment Gateway Integration Guide for React Native

Version Requirements:
  • React Native Version: ≥ 0.72.3
  • React Native SDK Version: 3.0.0
This guide will help you integrate PhonePe payment gateway in your React Native application.

1Installation


npm i react-native-phonepe-pg -f
npm i -f

2Platform Configuration

Android Setup

Add the following to your project level build.gradle:


allprojects { 
    repositories { 
        google() 
        maven { 
            url "https://phonepe.mycloudrepo.io/public/repositories/phonepe-intentsdk-android" 
        } 
    } 
}

iOS Setup

1. Add URL schemes to Info.plist:


<key>LSApplicationQueriesSchemes</key> 
<array> 
    <string>ppemerchantsdkv1</string>  
    <string>ppemerchantsdkv2</string> 
    <string>ppemerchantsdkv3</string>
    <string>paytmmp</string>
    <string>gpay</string>
</array>

2. Add URL Type for deep linking in Xcode

URL Scheme Requirements:
  • Only alphanumeric characters allowed
  • Special characters limited to dots and hyphens
  • Must start with alphabets

3. Install iOS dependencies:


cd ios 
pod install

4. Add deep linking handler to AppDelegate.m:


- (BOOL)application:(UIApplication *)app openURL:(NSURL *)url options:(NSDictionary *)options 
{ 
    NSMutableDictionary *userInfo = [[NSMutableDictionary alloc] init]; 
    [userInfo setObject:options forKey:@"options"];    
    [userInfo setObject:url forKey:@"openUrl"]; 
    [[NSNotificationCenter defaultCenter] postNotificationName: @"ApplicationOpenURLNotification" object:nil userInfo:userInfo]; 
    return YES; 
}

3Configuration

Create a configuration file (api/server.js):


// PhonePe Configuration
const ENV = 'UAT';  // or 'PROD' for production
const BASE_URL = 'https://api-preprod.phonepe.com/apis/pg-sandbox';
const CLIENT_ID = 'YOUR_CLIENT_ID';
const CLIENT_SECRET = 'YOUR_CLIENT_SECRET';
const CLIENT_VERSION = 1;
const MERCHANT_ID = 'YOUR_MERCHANT_ID';

// Token cache
let tokenCache = {
    access_token: null,
    expires_at: 0
};

4Auth Token Generation


export const generateAuthToken = async () => {
    try {
        const formData = new URLSearchParams();
        formData.append('client_id', CLIENT_ID);
        formData.append('client_secret', CLIENT_SECRET);
        formData.append('client_version', CLIENT_VERSION.toString());
        formData.append('grant_type', 'client_credentials');

        const response = await fetch(`${BASE_URL}/v1/oauth/token`, {
            method: 'POST',
            headers: {
                'Content-Type': 'application/x-www-form-urlencoded',
                'Accept': 'application/json'
            },
            body: formData.toString()
        });

        const data = await response.json();
        if (data.access_token) {
            tokenCache = {
                access_token: data.access_token,
                expires_at: data.expires_at
            };
            return data.access_token;
        }
        throw new Error('No access token in response');
    } catch (error) {
        throw error;
    }
};

5Order Creation


export const createOrder = async (amount) => {
    try {
        const token = tokenCache.access_token;
        if (!token) {
            throw new Error('No auth token available');
        }

        const merchantOrderId = `ORDER${Date.now()}${Math.random().toString(36).slice(2, 6)}`;

        const orderPayload = {
            merchantOrderId: merchantOrderId,
            amount: amount,
            expireAfter: 1200,
            metaInfo: {
                udf1: "React Native App",
                udf2: merchantOrderId
            },
            paymentFlow: {
                type: "PG_CHECKOUT"
            }
        };

        const response = await fetch(`${BASE_URL}/checkout/v2/sdk/order`, {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json',
                'Authorization': `O-Bearer ${token}`,
            },
            body: JSON.stringify(orderPayload)
        });

        const data = await response.json();
        return {
            orderId: data.orderId,
            token: data.token,
            merchantOrderId: merchantOrderId,
            state: data.state
        };
    } catch (error) {
        throw error;
    }
};

API Reference

1. Generate Auth Token

POST Endpoint: /v1/oauth/token

Base URL: https://api-preprod.phonepe.com/apis/pg-sandbox (UAT)

Request Headers:

{
    'Content-Type': 'application/x-www-form-urlencoded',
    'Accept': 'application/json'
}
Request Body:

{
    client_id: 'YOUR_CLIENT_ID',
    client_secret: 'YOUR_CLIENT_SECRET',
    client_version: '1',
    grant_type: 'client_credentials'
}
Success Response:

{
    "access_token": "eyJhbGciOiJIUzI1...",
    "token_type": "Bearer",
    "expires_in": 3600,
    "expires_at": 1679831543000
}
Error Response:

{
    "error": "invalid_client",
    "error_description": "Invalid client credentials"
}

2. Create Order

POST Endpoint: /checkout/v2/sdk/order

Request Headers:

{
    'Content-Type': 'application/json',
    'Authorization': 'O-Bearer YOUR_ACCESS_TOKEN'
}
Request Body:

{
    "merchantOrderId": "ORDER123456789",
    "amount": 100,                    // Amount in paise

        "udf1": "React Native App",
        "udf2": "ORDER123456789"
    },
    "paymentFlow": {
        "type": "PG_CHECKOUT"
    }
}
Success Response:

{
    "success": true,
    "code": "SUCCESS",
    "message": "Order created successfully",

        "orderId": "PGTXN123456789",
        "token": "TOKEN123456789",

Error Response:

Important Notes:

  • All amounts are in paise (1 INR = 100 paise)

  • Store sensitive credentials securely
  • Implement webhook handling for payment status updates

Example Code for Client Side

The payment flow implementation is as follows: 


                import React, { useEffect, useState } from 'react';
import { View, Text, TouchableOpacity, StyleSheet, Alert } from 'react-native';
import PhonePePaymentSDK from 'react-native-phonepe-pg';
import { generateAuthToken, createOrder } from '../../api/server';

const PhonePePayment = () => {
    const [isSDKInitialized, setIsSDKInitialized] = useState(false);
    const [isLoading, setIsLoading] = useState(false);

    useEffect(() => {
        initializeSDKAndToken();
    }, []);

    const initializeSDKAndToken = async () => {
        try {
            // Initialize SDK
            const sdkResult = await PhonePePaymentSDK.init(
                "SANDBOX",
                "SAHILSERVICEUAT",
                `FLOW_${Date.now()}`,
                true
            );
            console.log("SDK Initialization Result:", sdkResult);
            setIsSDKInitialized(true);

            // Generate auth token
            await generateAuthToken();
        } catch (error) {
            console.error("Initialization Error:", error);
            Alert.alert("Error", "Failed to initialize payment system");
        }
    };

    const startPayment = async () => {
        if (!isSDKInitialized) {
            Alert.alert("Error", "Payment system not initialized");
            return;
        }

        setIsLoading(true);
        try {
            // Create order
            const orderData = await createOrder(100); // ₹1 = 100 paise
            console.log('Order created:', orderData);

            // Prepare transaction request
            const requestBody = {
                orderId: orderData.orderId,
                merchantId: "SAHILSERVICEUAT",
                merchantOrderId: orderData.merchantOrderId,
                token: orderData.token,
                amount: 100,
                paymentMode: {
                    type: "PAY_PAGE",
                    paymentModeConfig: {
                        enabledPaymentModes: [
                            {
                                type: "UPI_INTENT"
                            },
                            {
                                type: "UPI_COLLECT"
                            },
                            {
                                type: "UPI_QR"
                            },
                            {
                                type: "CARD",
                                cardTypes: ["DEBIT_CARD", "CREDIT_CARD"]
                            }
                        ]
                    }
                },
                redirectUrl: "https://webhook.site/redirect-url",
                callbackUrl: "https://webhook.site/3da7dced-4e73-46b8-8760-8fa47482d359",
                deviceContext: {
                    deviceOS: "ANDROID"
                }
            };

            console.log('Starting transaction with:', requestBody);

            // Start transaction
            const response = await PhonePePaymentSDK.startTransaction(
                JSON.stringify(requestBody),
                null
            );
            console.log("Transaction Response:", response);
            
            handlePaymentStatus(response);

        } catch (error) {
            console.error("Payment Error:", error);
            Alert.alert(
                "Error",
                error.message || "Payment failed. Please try again."
            );
        } finally {
            setIsLoading(false);
        }
    };

    const handlePaymentStatus = (response) => {
        switch (response.status) {
            case "SUCCESS":
                Alert.alert("Success", "Payment completed successfully!");
                break;
            case "FAILURE":
                Alert.alert("Failed", "Payment failed. Please try again.");
                break;
            case "CANCELLED":
                Alert.alert("Cancelled", "Payment was cancelled.");
                break;
            default:
                Alert.alert("Status", `Unknown payment status: ${response.status}`);
        }
    };

    return (
        
            
                
                    {isLoading ? 'Processing...' : 'Pay ₹1 with PhonePe'}
                
            
        
    );
};
